# Haven Tile & Stone — Website Redesign & SEO Campaign
**Date:** 2026-05-20  
**Client:** Donovan — Haven Tile & Stone  
**Owner:** Dillon

---

## Overview

Full website redesign and ongoing SEO campaign for Haven Tile & Stone, a high-end tile and stone installation company serving luxury homeowners along the East Coast Carolina beach region. The goal is a visually stunning site that converts organic search traffic into leads — with different entry points (Google search results) routing visitors to the right conversion path for their intent.

---

## Visual Direction

**Aesthetic:** Light & Coastal Luxury  
**Palette:** Warm cream (`#FDFAF5`), sand (`#F5F0E8`), coastal teal (`#2A3F3C`), brushed gold (`#C8A96A`), charcoal (`#1C1C1C`)  
**Typography:** Playfair Display (headings/serif), DM Sans (body/UI)  
**Photography style:** Full-bleed natural light, real stone/tile textures, coastal interiors  
**Feel:** High-end beach house — airy, clean, sophisticated but relaxed

---

## Platform

**WordPress** — chosen for best-in-class local SEO tooling (Yoast/RankMath), fast iteration on new location/service pages, and Donovan's ability to make basic updates independently.

**Hosting:** Currently on GoDaddy. Domain (haventileandstone.com) stays at GoDaddy as registrar. Hosting may migrate to SiteGround or WP Engine for speed — decision deferred. Cloudflare CDN (free tier) to be added regardless.

## Conversion Paths (Multi-Intent)

The site must serve multiple search intents — the entry page matches the visitor's intent, each with the appropriate CTA:

| Search Intent | Entry Page | Primary CTA |
|---|---|---|
| "tile contractor Ocean Isle" | Homepage or location page | Request Free Quote form |
| "bathroom remodel Holden Beach" | Bathroom + location page | Request Free Quote form |
| "tile contractor near me" | Homepage | Phone/text number |
| "luxury tile portfolio NC" | Portfolio/Gallery page | Browse work → Contact |
| "schedule tile consultation" | Contact/Consultation page | Book a consultation |

All pages carry the phone number prominently. The quote form captures: name, phone, email, service type, location, project description.

## SEO Strategy

**On-page:** Each page targets a primary keyword + 2-3 secondary keywords. Title tags, H1s, meta descriptions, image alt text, and schema markup (LocalBusiness, Service) all optimized.

**Local SEO:** Google Business Profile optimization, NAP consistency across all pages and directories, location-specific content on every location page (landmarks, neighborhoods, context).

**Content:** Service pages written to answer buyer-intent questions. Location pages go beyond thin content — each includes local context, project examples from that area, and unique copy.

**Technical:** Fast hosting + Cloudflare CDN, Core Web Vitals optimized, mobile-first, XML sitemap, robots.txt, structured data.

**Ongoing campaign:** Monthly blog posts targeting long-tail keywords (e.g., "best tile for pool deck salt air Carolina"), link building, Google Business Profile posting, review generation.
